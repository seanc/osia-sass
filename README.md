# osia-sass
> SASS Compiler for Osia

## Installation
```shell
$ npm install osia-sass
```

## Usage
```javascript
import osia from 'osia';
import sass from 'osia-sass';

osia.task('build', () => 
  osia.open('foo.scss')
    .then(sass())
    .then(bar())
);
```
